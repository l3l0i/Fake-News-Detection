# Fake News Detection using Machine Learning

## Project Overview
This project builds a Machine Learning model to classify news articles as **Real or Fake**.

## Technologies Used
- Python
- Scikit-learn
- Pandas
- Seaborn
- Matplotlib
- Google Colab

## Model Workflow
1. Data preprocessing
2. Text vectorization using TF-IDF
3. Model training
4. Model evaluation using Accuracy and Confusion Matrix
5. Saving the trained model

## Model Performance
Accuracy: **98.9%**

Confusion Matrix results show very low misclassification.

## Files
- `fake_news_model.ipynb` : main notebook
- `fake_news_model.pkl` : saved trained model
- `dataset` : training dataset
